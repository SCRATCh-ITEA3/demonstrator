## Run the demonstrator from the docker hub
```shell
docker pull scratchitea3/demonstrator
docker run scratchitea3/demonstrator
```

## Build it locally
As docker hub sometimes delays the build, its adviced to test the software locally.

```shell
docker-compose up
docker-compose run
```
